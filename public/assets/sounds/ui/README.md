# UI / menu sounds

Drop these `.ogg` files here (referenced by `src/game/audio/uiSounds.ts`):

- `69_ui-generic_button_click_01.wav` — any menu button press (not movement)
- `81_ui-store_buy_01.wav` — successful item purchase
- `10_scn1_btn_1.wav` — tried to buy something you can't afford

Missing files fail silently, so the game still runs without them.
